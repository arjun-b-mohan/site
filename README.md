# Personal Website

Arjun's personal website - a simple, clean HTML-based personal  site.

## Features

- **Home Page**: Bio, LinkedIn connection, and downloadable resume
- **Books**: List of favorite books recently read
- **Writing**: Collection of blog posts and articles

## Setup

1. **Personalize the content**:
   - Edit `index.html` to add your bio and LinkedIn URL
   - Update `books.html` with your favorite books
   - Update `writing.html` with your blog posts and articles

2. **Add your resume**:
   - Place your resume PDF in the root directory as `resume.pdf`

3. **Customize styling** (optional):
   - Edit `style.css` to adjust colors, fonts, or layout

## Files

- `index.html` - Main landing page with bio and links
- `books.html` - Books you've read
- `writing.html` - Your blog posts and articles
- `style.css` - Simple, responsive styling

## Deployment

This is plain HTML/CSS, so you can deploy it anywhere:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply upload the files and you're done!
